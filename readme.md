### Node Talk - README (For Windows Distribution)

## Overview

Node Talk is a CUI chat application for LAN, running as a single `.exe` file for Windows.
It is a lightweight tool that can be used instantly within a local network without relying on email or cloud services.

---

## Included Files

* `NodeTalk.exe` (executable file for Windows)
* `NodeTalk-jp.exe` (executable file for Windows, Japanese version)

---

## How to Use

1. Transfer `NodeTalk.exe` to a Windows PC by any method
2. Double-click to run
3. After startup, the following menu will be displayed:

```
=== Node Talk ===
1. Start in Server Mode
2. Connect as Client
3. Exit
Please enter a number:
```

---

## Key Features

* Choose "Server" or "Client" mode at startup
* Connect using local IP address
* Chat after entering a nickname
* TCP communication using Winsock2
* Simultaneous send/receive with multithreading
* Safe exit with Ctrl+C

---

## System Requirements

* Windows 10 / 11 or later
* No runtime environment such as .NET required

---

## Use Cases

* Communication within a school LAN
* Offline emergency communication
* Usable even without internet connection
* Run directly from USB or DVD
